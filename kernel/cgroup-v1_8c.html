int cgroup_transfer_tasks ( struct cgroup *  to,
struct cgroup *  from 
)

cgroup_trasnsfer_tasks - move tasks from one cgroup to another : cgroup to which the tasks will be moved : cgroup in which the tasks currently reside

Locking rules between cgroup_post_fork() and the migration path guarantee that, if a task is forking while being migrated, the new child is guaranteed to be either visible in the source cgroup after the parent's migration is complete or put into the target cgroup. No task can slip out of migration through forking.
