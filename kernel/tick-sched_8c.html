int tick_check_oneshot_change ( int  allow_nohz)

Check, if a change happened, which makes oneshot possible.

Called cyclic from the hrtimer softirq (driven by the timer softirq) allow_nohz signals, that we can switch into low-res nohz mode, because high resolution timers are disabled (either compile or runtime). Called with interrupts disabled.
