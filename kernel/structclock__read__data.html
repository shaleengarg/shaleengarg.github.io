Detailed Description

struct clock_read_data - data required to read from sched_clock()

: sched_clock() value at last update : Clock cycle value at last update. : Bitmask for two's complement subtraction of non 64bit clocks. : Current clock source (or dummy source when suspended). : Multipler for scaled math conversion. : Shift value for scaled math conversion.

Care must be taken when updating this structure; it is read by some very hot code paths. It occupies <=40 bytes and, when combined with the seqcount used to synchronize access, comfortably fits into a 64 byte cache line.
