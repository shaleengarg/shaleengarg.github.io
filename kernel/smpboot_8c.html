smpboot_thread_fn - percpu hotplug thread loop function : thread data pointer

Checks for thread stop and park conditions. Calls the necessary setup, cleanup, park and unpark functions for the registered thread.

Returns 1 when the thread should exit, 0 otherwise.

smpboot_update_cpumask_percpu_thread - Adjust which per_cpu hotplug threads stay parked : Hotplug thread descriptor : Revised mask to use

The cpumask field in the smp_hotplug_thread must not be updated directly by the client, but only by calling this function. This function can only be called on a registered smp_hotplug_thread.
