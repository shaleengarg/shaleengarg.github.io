irq_gc_mask_disable_and_ack_set - Mask and ack pending interrupt : irq_data

This generic implementation of the irq_mask_ack method is for chips with separate enable/disable registers instead of a single mask register and where a pending interrupt is acknowledged by setting a bit.

Note: This is the only permutation currently used. Similar generic functions should be added here if other permutations are required.
