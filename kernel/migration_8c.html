irq_fixup_move_pending - Cleanup irq move pending from a dying CPU : Interrupt descpriptor to clean up : If set clear the move pending bit unconditionally. If not set, clear it only when the dying CPU is the last one in the pending mask.

Returns true if the pending bit was set and the pending mask contains an online CPU other than the dying CPU.
