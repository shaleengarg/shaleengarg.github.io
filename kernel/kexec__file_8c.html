int kexec_add_buffer ( struct kexec_buf *  kbuf)

kexec_add_buffer - place a buffer in a kexec segment : Buffer contents and memory parameters.

This function assumes that kexec_mutex is held. On successful return, ->mem will have the physical address of the buffer in memory.

Return: 0 on success, negative errno on error.
