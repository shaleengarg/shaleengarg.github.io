timecounter_read_delta - get nanoseconds since last call of this function : Pointer to time counter

When the underlying cycle counter runs over, this will be handled correctly as long as it does not run over more than once between calls.

The first call to this function for a new time counter initializes the time tracking and returns an undefined result.
