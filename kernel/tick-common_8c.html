int tick_broadcast_oneshot_control ( enum tick_broadcast_state  state)

tick_broadcast_oneshot_control - Enter/exit broadcast oneshot mode : The target state (enter/exit)

The system enters/leaves a state, where affected devices might stop Returns 0 on success, -EBUSY if the cpu is used to broadcast wakeups.

Called with interrupts disabled, so clockevents_lock is not required here because the local clock event device cannot go away under us.
