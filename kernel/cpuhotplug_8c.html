irq_migrate_all_off_this_cpu - Migrate irqs away from offline cpu

The current CPU has been marked offline. Migrate IRQs off this CPU. If the affinity settings do not allow other CPUs, force them onto any available CPU.

Note: we must iterate over all IRQs, whether they have an attached action structure or not, as we need to get chained interrupts too.
