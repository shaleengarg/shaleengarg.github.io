create_filter - create a filter for a trace_event_call : trace_event_call to create a filter for : filter string : remember and enable detailed error in filter : out param for created filter (always updated on return)

Creates a filter for with . If is true, is copied and recorded in the new filter.

On success, returns 0 and * points to the new filter. On failure, returns -errno and * may point to NULL or to a new filter. In the latter case, the returned filter contains error information if is true and the caller is responsible for freeing it.
