Detailed Description

struct klp_ops - structure for tracking registered ftrace ops structs

A single ftrace_ops is shared between all enabled replacement functions (klp_func structs) which have the same old_addr. This allows the switch between function versions to happen instantaneously by updating the klp_ops struct's func_stack list. The winner is the klp_func at the top of the func_stack (front of the list).

: node for the global klp_ops list : list head for the stack of klp_func's (active func is on top) : registered ftrace ops struct
