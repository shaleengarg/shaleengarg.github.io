static int wait_task_stopped ( struct wait_opts *  wo,
int  ptrace,
struct task_struct *  p 
)
static

wait_task_stopped - Wait for TASK_STOPPED or TASK_TRACED : wait options : is the wait for ptrace : task to wait for

Handle sys_wait4() work for p in state TASK_STOPPED or TASK_TRACED.

CONTEXT: read_lock(&tasklist_lock), which is released if return value is non-zero. Also, grabs and releases ->sighand->siglock.

RETURNS: 0 if wait condition didn't exist and search for other wait conditions should continue. Non-zero return, -errno on failure and 's pid on success, implies that tasklist_lock is released and wait condition search should terminate.
