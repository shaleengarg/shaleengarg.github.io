Detailed Description

struct gcov_fn_info - profiling meta data per function : object file-unique function identifier : function checksum : number of values per counter type belonging to this function

This data is generated by gcc during compilation and doesn't change at run-time.

struct gcov_fn_info - profiling meta data per function : comdat key : unique ident of function : function lineo_checksum : function cfg checksum : instrumented counters

This data is generated by gcc during compilation and doesn't change at run-time.

Information about a single function. This uses the trailing array idiom. The number of counters is determined from the merge pointer array in gcov_info. The key is used to detect which of a set of comdat functions was selected – it points to the gcov_info object of the object file containing the selected comdat function.
