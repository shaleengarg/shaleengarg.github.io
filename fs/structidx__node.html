Detailed Description

struct idx_node - hold index nodes during index tree traversal. : list : index in parent (slot number of this indexing node in the parent indexing node) : all keys in this indexing node have to be less or equivalent to this key : index node (8-byte aligned because all node structures must be 8-byte aligned)
