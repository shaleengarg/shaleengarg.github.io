◆ v9fs_dec_count()

static void v9fs_dec_count ( struct inode *  inode)
static

v9fs_dec_count - helper functon to drop i_nlink.

If a directory had nlink <= 2 (including . and ..), then we should not drop the link count, which indicates the underlying exported fs doesn't maintain nlink accurately. e.g.

  • overlayfs sets nlink to 1 for merged dir
  • ext4 (with dir_nlink feature enabled) sets nlink to 1 if a dir has more than EXT4_LINK_MAX (65000) links.

: inode whose nlink is being dropped
