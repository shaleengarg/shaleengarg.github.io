Detailed Description

struct ubifs_ch - common header node. : UBIFS node magic number (UBIFS_NODE_MAGIC) : CRC-32 checksum of the node header : sequence number : full node length : node type : node group type : reserved for future, zeroes

Every UBIFS node starts with this common part. If the node has a key, the key always goes next.
