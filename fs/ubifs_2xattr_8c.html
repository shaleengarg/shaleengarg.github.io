ubifs_evict_xattr_inode - Evict an xattr inode. : UBIFS file-system description object : xattr inode number

When an inode that hosts xattrs is being removed we have to make sure that cached inodes of the xattrs also get removed from the inode cache otherwise we'd waste memory. This function looks up an inode from the inode cache and clears the link counter such that iput() will evict the inode.
