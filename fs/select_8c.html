◆ poll_select_set_timeout()

int poll_select_set_timeout ( struct timespec64 *  to,
time64_t  sec,
long  nsec 
)

poll_select_set_timeout - helper function to setup the timeout value : pointer to timespec64 variable for the final timeout : seconds (from user space) : nanoseconds (from user space)

Note, we do not use a timespec for the user space value here, That way we can use the function for timeval and compat interfaces as well.

Returns -EINVAL if sec/nsec are not normalized. Otherwise 0.
