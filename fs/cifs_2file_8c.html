cifs_readdata_to_iov - copy data from pages in response to an iovec : the readdata response with list of pages holding data : destination for our data

This function copies data from a list of pages in a readdata response into an array of iovecs. It will first calculate where the data should go based on the info in the readdata and then copy the data into that spot.
