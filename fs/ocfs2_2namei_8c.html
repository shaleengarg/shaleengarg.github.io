ocfs2_prepare_orphan_dir() - Prepare an orphan directory for insertion of an orphan. : ocfs2 file system : Orphan dir inode - returned locked! : Actual block number of the inode to be inserted into orphan dir. : dir lookup result, to be passed back into functions like ocfs2_orphan_add

Returns zero on success and the ret_orphan_dir, name and lookup fields will be populated.

Returns non-zero on failure.
