Detailed Description

struct ubifs_wbuf - UBIFS write-buffer. : UBIFS file-system description object : write-buffer (of min. flash I/O unit size) : logical eraseblock number the write-buffer points to : write-buffer offset in this logical eraseblock : number of bytes available in the write-buffer : number of used bytes in the write-buffer : write-buffer size (in [->min_io_size, ->max_write_size] range) : journal head the mutex belongs to (note, needed only to shut lockdep up by 'mutex_lock_nested()). : write-buffer synchronization callback : serializes write-buffer I/O : serializes , , , , , and fields : write-buffer timer : non-zero if this write-buffer does not have a timer : non-zero if the timer expired and the wbuf needs sync'ing : points to the next position of the following inode number : stores the inode numbers of the nodes which are in wbuf

The write-buffer synchronization callback is called when the write-buffer is synchronized in order to notify how much space was wasted due to write-buffer padding and how much free space is left in the LEB.
