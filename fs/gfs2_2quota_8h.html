int gfs2_quota_check ( struct gfs2_inode *  ip,
kuid_t  uid,
kgid_t  gid,
struct gfs2_alloc_parms *  ap 
)

gfs2_quota_check - check if allocating new blocks will exceed quota : The inode for which this check is being performed : The uid to check against : The gid to check against : The allocation parameters. ap->target contains the requested blocks. ap->min_target, if set, contains the minimum blks requested.

Returns: 0 on success. min_req = ap->min_target ? ap->min_target : ap->target; quota must allow atleast min_req blks for success and ap->allowed is set to the number of blocks allowed

-EDQUOT otherwise, quota violation. ap->allowed is set to number of blocks available.
