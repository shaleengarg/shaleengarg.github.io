wchar_t cifs_toupper ( wchar_t  in)

cifs_toupper - convert a wchar_t from lower to uppercase : character to convert from lower to uppercase

This function consults the static tables above to convert a wchar_t from lower to uppercase. In the event that there is no mapping, the original "in" character is returned.
