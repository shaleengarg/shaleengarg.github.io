◆ nfs4_ff_layout_prepare_ds()

struct nfs4_pnfs_ds* nfs4_ff_layout_prepare_ds ( struct pnfs_layout_segment *  lseg,
u32  ds_idx,
bool  fail_return 
)

nfs4_ff_layout_prepare_ds - prepare a DS connection for an RPC call : the layout segment we're operating on : index of the DS to use : return layout on connect failure?

Try to prepare a DS connection to accept an RPC call. This involves selecting a mirror to use and connecting the client to it if it's not already connected.

Since we only need a single functioning mirror to satisfy a read, we don't want to return the layout if there is one. For writes though, any down mirror should result in a LAYOUTRETURN. is how we distinguish between the two cases.

Returns a pointer to a connected DS object on success or NULL on failure.
