◆ kernfs_pin_sb()

struct super_block* kernfs_pin_sb ( struct kernfs_root *  root,
const void *  ns 
)

kernfs_pin_sb: try to pin the superblock associated with a kernfs_root : the kernfs_root in question : the namespace tag

Pin the superblock so the superblock won't be destroyed in subsequent operations. This can be used to block ->kill_sb() which may be useful for kernfs users which dynamically manage superblocks.

Returns NULL if there's no superblock associated to this kernfs_root, or -EINVAL if the superblock is being freed.
