We read each buffer asynchronously and when all buffers are read in, our io completion handler ntfs_end_buffer_read_async(), if required, automatically applies the mst fixups to the page before finally marking it uptodate and unlocking it.

We only enforce allocated_size limit because i_size is checked for in generic_file_read().

Return 0 on success and -errno on error.

Contains an adapted version of fs/buffer.cblock_read_full_page().
