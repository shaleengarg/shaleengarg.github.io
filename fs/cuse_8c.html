cuse_parse_one - parse one key=value pair : i/o parameter for the current position : points to one past the end of the packed string : out parameter for key : out parameter for value

* points to packed strings - "key0=val0\0key1=val1\0" which ends at - 1. This function parses one pair and set * to the start of the key and * to the start of the value. Note that the original string is modified such that the key string is terminated with '\0'. * is updated to point to the next string.

RETURNS: 1 on successful parse, 0 on EOF, -errno on failure.
