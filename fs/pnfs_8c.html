const struct pnfs_layout_range *  recall_range,
u32  seq 
)

pnfs_mark_matching_lsegs_invalid - tear down lsegs or mark them for later : layout header containing the lsegs : list head where doomed lsegs should go : optional recall range argument to match (may be NULL) : only invalidate lsegs obtained prior to this sequence (may be 0)

Walk the list of lsegs in the layout header, and tear down any that should be destroyed. If "recall_range" is specified then the segment must match that range. If "seq" is non-zero, then only match segments that were handed out at or before that sequence.

Returns number of matching invalid lsegs remaining in list after scanning it and purging them.
