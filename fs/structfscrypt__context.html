Detailed Description

Encryption context for inode

Protector format: 1 byte: Protector format (1 = this version) 1 byte: File contents encryption mode 1 byte: File names encryption mode 1 byte: Flags 8 bytes: Master Key descriptor 16 bytes: Encryption Key derivation nonce
