befs_nls2utf - Convert NLS string to utf8 encodeing : Superblock : Input string buffer in NLS format : Length of input string in bytes : The output string in UTF-8 format : Length of the output buffer

Converts input string , which is in the format of the loaded NLS map, into a utf8 string.

The destination string is allocated by this function and the caller is responsible for freeing it with kfree()

On return, * is the length of in bytes.

On success, the return value is the number of utf8 characters written to the output buffer .

On Failure, a negative number coresponding to the error code is returned.
