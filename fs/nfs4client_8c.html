nfs4_update_server - Move an nfs_server to a different nfs_client

: represents FSID to be moved : new end-point's hostname : new end-point's socket address : size of "sap" : net namespace

The nfs_server must be quiescent before this function is invoked. Either its session is drained (NFSv4.1+), or its transport is plugged and drained (NFSv4.0).

Returns zero on success, or a negative errno value.
