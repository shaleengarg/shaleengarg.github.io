fat_parse_long - Parse extended directory entry.

This function returns zero on success, negative value on error, or one of the following:

PARSE_INVALID - Directory entry is invalid. PARSE_NOT_LONGNAME - Directory entry does not contain longname. PARSE_EOF - Directory has no more entries.
