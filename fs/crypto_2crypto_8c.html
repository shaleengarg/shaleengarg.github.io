void fscrypt_release_ctx ( struct fscrypt_ctx *  ctx)

fscrypt_release_ctx() - Releases an encryption context : The encryption context to release.

If the encryption context was allocated from the pre-allocated pool, returns it to that pool. Else, frees it.

If there's a bounce page in the context, this frees that.
