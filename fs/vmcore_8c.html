alloc_elfnotes_buf - allocate buffer for ELF note segment in vmalloc memory

: size of buffer

If CONFIG_MMU is defined, use vmalloc_user() to allow users to mmap the buffer to user-space by means of remap_vmalloc_range().

If CONFIG_MMU is not defined, use vzalloc() since mmap_vmcore() is disabled and there's no need to allow users to mmap the buffer.

copy_notes_elf32 - copy ELF note segments in a given buffer

: ELF header : buffer into which ELF note segments are copied

This function is used to copy ELF note segment in the 1st kernel into the buffer in the 2nd kernel. It is assumed that size of the buffer is equal to or larger than sum of the real ELF note segment headers and data.

It is assumed that program headers with PT_NOTE type pointed to by has already been updated by update_note_header_size_elf32 and each of PT_NOTE program headers has actual ELF note segment size in its p_memsz member.
