pagevec_lookup_range - gang pagecache lookup : Where the resulting pages are placed : The address_space to search : The starting page index : The final page index

pagevec_lookup_range() will search for & return a group of up to PAGEVEC_SIZE pages in the mapping starting from index and upto index (inclusive). The pages are placed in . pagevec_lookup() takes a reference against the pages in .

The search returns a group of mapping-contiguous pages with ascending indexes. There may be holes in the indices due to not-present pages. We also update to index the next page for the traversal.

pagevec_lookup_range() returns the number of pages which were found. If this number is smaller than PAGEVEC_SIZE, the end of specified range has been reached.
