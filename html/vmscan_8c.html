◆ shrink_slab()

static unsigned long shrink_slab ( gfp_t  gfp_mask,
int  nid,
struct mem_cgroup *  memcg,
int  priority 
)
static

shrink_slab - shrink slab caches : allocation context : node whose slab caches to target : memory cgroup whose slab caches to target : the reclaim priority

Call the shrink functions to age shrinkable caches.

is passed along to shrinkers with SHRINKER_NUMA_AWARE set, unaware shrinkers will receive a node id of 0 instead.

specifies the memory cgroup to target. If it is not NULL, only shrinkers with SHRINKER_MEMCG_AWARE set will be called to scan objects from the memory cgroup specified. Otherwise, only unaware shrinkers are called.

is sc->priority, we take the number of objects and >> by priority in order to get the scan target.

Returns the number of reclaimed slab objects.
